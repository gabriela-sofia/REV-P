# QGIS digitization steps

1. Open the QGIS project.
2. Load `recife_risk_areas_context.geojson`.
3. Load the Charter PNG for visual reference only.
4. Create polygon layer `patch_boundary_REC_00019` from a verified patch source.
5. Create polygon layer `event_polygon_REC_2022_05_24_30` from verified observed evidence.
6. Save as GeoJSON with explicit CRS.
7. Fill the original FILL_THIS files.
8. Run v2ba validation.
