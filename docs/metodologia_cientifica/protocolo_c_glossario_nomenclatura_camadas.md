# Glossario de nomenclatura de camadas do Protocolo C

## Feições poligonais de deslizamento fotointerpretadas

**Nome público/conceitual:** Feições poligonais de deslizamento fotointerpretadas

**Alias técnico:** PET_CPRM_DESLIZAMENTO_AREA_FOTOINTERPRETADA

**Nome original bruto:** Cicatriz_Area_A.shp

**Razão da padronização:** o nome original do arquivo é mantido apenas como rastreabilidade de fonte. Em texto público, registries e decisões metodológicas, a camada deve ser descrita pelo que ela documenta de forma conservadora: feições poligonais de deslizamento obtidas por fotointerpretação.

**Alerta metodológico:** esta padronização não implica data específica, evento específico, ground truth operacional, label supervisionado, negativo formal ou permissão de treino. A camada continua sujeita aos gates do Protocolo C, incluindo bloqueio temporal quando não houver evidência documental suficiente.

## Regra de uso

Use o alias técnico em identificadores públicos, tabelas e registries. Use o nome público em texto narrativo. Use o nome original bruto somente em campos de proveniência ou rastreabilidade, como `source_layer_original_name`, `original_filename`, `raw_source_filename` ou `provenance_original_layer`.
