# Protocolo C - resolucao de features dos controles v1ks-v1kz

A sequencia separa o gargalo numerico em coordenada, probe GEE, aquisicao real de patch, features S2/DEM/DINO congelado e recheck dos gates de controle.

Nenhuma etapa transforma controle em negativo formal, cria label operacional ou treina modelo.
