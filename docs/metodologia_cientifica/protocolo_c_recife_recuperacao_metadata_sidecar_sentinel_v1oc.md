# Protocolo C - Recife recuperacao por metadata/sidecar v1oc

O script le sidecars locais e, quando raster local existe, apenas tags/header metadata via rasterio se disponivel.

Nenhum pixel e processado. Conteudo bruto local nao e copiado para outputs publicos.

Sem arquivo local ou sidecar, o status permanece LOCAL_METADATA_NOT_AVAILABLE.
