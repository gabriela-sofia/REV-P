# REV-P v1up -- Precommit Public Gate (PowerShell)

## Purpose

A PowerShell gate script that runs before staging to block policy violations.
Generated by `revp_v1up_precommit_gate_generator.py`.

## Location

`scripts/git/revp_precommit_public_gate.ps1`

## Usage

```powershell
powershell -File scripts/git/revp_precommit_public_gate.ps1
```

Returns exit code 0 if clean, exit code 1 if violations found.

## What is blocked

| Rule | Condition |
|------|-----------|
| Large file | Staged file > 50 MB |
| Never-version directory | `data/external_raw/`, `data/external_cache/`, `local_runs/`, `local_only/` |
| Known large local-only files | `formal_negative_evidence_intake_registry.csv`, `formal_negative_candidate_decision_audit.csv` |
| Absolute Windows path | Content matches `[A-Za-z]:\...` |
| local_runs reference | Content contains `local_runs` |
| Forbidden terminology | e.g. "ground truth operacional", "c4 aberto" |
| Forbidden guardrail flag | e.g. `can_train_model,true` |

## Claudete false-positive protection

The script uses word-boundary matching. "Claudete" is never flagged as "Claude".

## Installation as git hook (manual)

This script is NOT installed automatically. To install:

```powershell
Copy-Item scripts/git/revp_precommit_public_gate.ps1 .git/hooks/pre-commit
```

## Warnings (non-blocking)

Files between 10MB and 50MB emit a `[WARN]` message but do not block the commit.
