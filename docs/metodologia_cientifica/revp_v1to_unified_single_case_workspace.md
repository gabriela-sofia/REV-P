# v1to — Unified Single-Case Workspace

## Objetivo

Workspace única e legível por caso (resumo, evidência externa, hidromet, papel do DINO, patch/evento, blockers, riscos de overclaim, próximos passos e status review-only).

## Resultado
Workspaces: 9. Seções: 81.

## Limitação

Permite ler um caso sem abrir dezenas de CSVs. Não cria label/target/ground truth, não promove C3, não abre C4, não usa hidromet/DINO como prova.
