# v1ss — Protocol C Event Date Window Builder

## Objetivo

Construir janelas temporais (T-7 a T+1) para cada evento candidato com data parseavel. Fail-closed quando sem datas.

## Resultado

Janelas: 9. Parseadas: 9. Regioes: PET.

## Limitacao

Janela temporal compativel nao valida evento automaticamente.
