# v1tx — Case Dossier Exporter

## Objetivo

Exportar um dossie de evidencia legivel por caso a partir da revisao automatizada review-only (v1tn-v1tw).

## Resultado
Dossies: 9. Secoes: 72.

## Limitacao

DINO/hidromet sao contexto; ausencia nao e negativo. Sem C3 automatico, sem C4, sem ground truth/rotulo/target/negativo formal. Fonte externa exigida para afirmacao operacional.
