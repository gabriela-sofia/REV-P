# REV-P v1un — Public Staging Candidate Manifest

## Purpose

Combines v1ul policy decisions and v1um summary index to produce exact lists of
files recommended for staging and files excluded from staging.

## Outputs

- `datasets/repository_public_staging_candidate_manifest_v1un.csv` — files OK to stage
- `datasets/repository_public_staging_exclusion_manifest_v1un.csv` — files to exclude

## Actions

| Action | Meaning |
|--------|---------|
| INCLUDE | Safe to stage and version |
| EXCLUDE | Must not be staged |
| EXCLUDE_USE_SUMMARY | Full file excluded; use the public summary instead |
| REVIEW | Requires manual review before staging |

## Guardrails

Review-only. Large CSVs, raw data, cache, local_runs, and dirty findings are
excluded. Public summaries generated by v1um are included.
