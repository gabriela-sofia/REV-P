# Protocolo C - Recife inventario de fontes de data Sentinel v1oa

O inventario lista manifests, nomes de arquivo, sidecars e candidatos a metadata local onde uma data Sentinel REC poderia estar registrada.

O processo e metadata-only: nao le pixel, nao copia raster e nao transforma janela de evento em scene_date.

Fontes sem data explicita permanecem apenas como candidatos de recuperacao.
