# v1re — External Event/Patch Candidate Linker

## Objetivo

Ligar candidatos de evento externos (v1rd) a patches existentes como candidatos de link REVIEW-ONLY, por correspondencia de regiao/ameaca.

## Resultado

Status: EXTERNAL_INTAKE_WAITING_MANUAL_DOCUMENTS. Candidatos de link: 0 (0 review-only).

## Guardrails

Um link nunca e prova de evento e nunca e label. dino_validates_event=false. absence_as_negative=false. Nenhum target/ground truth operacional.
