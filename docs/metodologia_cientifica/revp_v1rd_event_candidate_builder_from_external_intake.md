# v1rd — Event Candidate Builder from External Intake

## Objetivo

Construir candidatos de evento review-only a partir do intake externo validado (v1rc). Sem intake valido, fica EXTERNAL_INTAKE_WAITING_MANUAL_DOCUMENTS.

## Resultado

Status: EXTERNAL_INTAKE_WAITING_MANUAL_DOCUMENTS. Candidatos de evento: 0.

## Guardrails

candidate_status=REVIEW_ONLY_EXTERNAL_CANDIDATE. dino_validates_event=false. absence_as_negative=false. Nenhum label/target/ground truth operacional.
