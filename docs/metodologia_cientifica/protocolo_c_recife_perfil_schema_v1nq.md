# Protocolo C - Recife perfil de schema v1nq

O perfil registra colunas, papeis e vocabularios reais observados nos CSVs oficiais baixados localmente.

Valores como vistoria_risco ou situacao sao auditados como vocabulario; nenhum valor isolado cria negativo formal.

Sem risco ou equivalente so pode seguir adiante se aparecer com fonte oficial, data, localidade, fenomeno e adjudicacao posterior.
