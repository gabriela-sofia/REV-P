# Protocolo C - Recife normalizacao de datas v1nu

Este bloco normaliza datas dos candidatos positivos oficiais de Recife e conserva 2021-2024 como contexto, com prioridade analitica para 2022.

Datas ausentes, parciais ou invalidas bloqueiam linkage temporal. Nenhuma data Sentinel e inferida a partir do evento.

can_create_operational_label=false e can_train_model=false em todos os outputs.
