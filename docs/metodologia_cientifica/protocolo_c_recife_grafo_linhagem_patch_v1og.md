# Protocolo C - Recife grafo de linhagem patch Sentinel v1og

O grafo conecta patch_id, aliases, manifests, nomes de asset e datas candidatas em modo metadata-only.

O output mostra onde a linhagem patch -> asset -> produto Sentinel -> scene_date quebra.

Nenhuma data e confirmada nesta etapa sem product/date vinculados ao patch.
