# Protocolo C - candidatos negativos derivados de inventario v1jw

v1jw so emite candidatos se v1jv confirmar ao menos uso parcial review-only do inventario. Os pontos sao deterministicos, dentro da moldura auditada, fora dos poligonos mapeados e fora dos buffers positivos.

Sem inventario completo, os candidatos permanecem review-only e `can_create_training_label=false`.
