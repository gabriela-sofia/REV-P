# REV-P v2cs - real external source seeding

Registry inicial de fontes externas reais para triagem documental, licenca,
descoberta manual de produtos e QA geoespacial futuro.

- `REAL_v2cs_CHARTER_751_PETROPOLIS`: International Charter Activation 751 - Flood and landslide in Brazil (DOCUMENTARY_SOURCE_REQUIRES_REVIEW)
- `REAL_v2cs_CHARTER_758_PERNAMBUCO_RECIFE`: International Charter Activation 758 - Landslides in Brazil (DOCUMENTARY_SOURCE_REQUIRES_REVIEW)
- `REAL_v2cs_COPERNICUS_EMS_ON_DEMAND`: Copernicus EMS On Demand Mapping (SOURCE_FAMILY_ONLY)
- `REAL_v2cs_COPERNICUS_GFM`: Copernicus Global Flood Monitoring / GFM (SOURCE_FAMILY_ONLY)
- `REAL_v2cs_SGB_CPRM_PREVENCAO_DESASTRES`: SGB/CPRM Geoportal Prevenção de Desastres (CONTEXTUAL_SOURCE_REQUIRES_REVIEW)
- `REAL_v2cs_DRM_RJ_CARTA_RISCO_PETROPOLIS`: Carta de Risco Petropolis - Portal de Dados Abertos RJ (CONTEXTUAL_SOURCE_REQUIRES_REVIEW)
- `REAL_v2cs_CURITIBA_DADOS_ABERTOS_IPPUC`: Portal de Dados Abertos de Curitiba / IPPUC (CONTEXTUAL_SOURCE_REQUIRES_REVIEW)

Todas as fontes permanecem review-only. Downloads e publicacao de arquivo bruto
ficam bloqueados enquanto licenca, produto, CRS, hash e metadados nao forem
confirmados.
