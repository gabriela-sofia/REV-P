# Charter 758 Recife controlled digitization

Use only the archived official Recife product. First georeference it in QGIS using verified control points visible in the product and an accepted CRS. Record every GCP and residual. Digitize only explicitly drawn observed landslide scars/effects. Save a Polygon/MultiPolygon GeoJSON and require human review. Do not use risk points, AOI, patch footprint or freehand assumptions.
