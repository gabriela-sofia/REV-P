# v2aw synthetic geometry intake examples

These files are **synthetic** and exist only for documentation and tests.

- They use placeholder ids (`EXAMPLE_PATCH_*`, `EXAMPLE_EVENT_*`) and never a real `patch_id` or `event_id`.
- Do **not** mix them with real data and do **not** feed them to v2av/v2au as if real.
- They show the accepted formats: bbox, WKT and GeoJSON.

Nothing here is a label, ground truth or training target.
